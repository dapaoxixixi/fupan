使用说明
1.自己注册韭研公社的账号填进去
2.板块复盘的成交额和流通市值抓取比较慢，需要等等。
3.连板天梯会抓取最近7个交易日的数据缓存到本地，第一次打开会比较慢
4.市场情绪里的数据16点后刷新是会保存在本地的，之后再看还是有的。第一次使用想要获取历史的数据的话需要token，要充钱的，不建议
